# Agent Instructions

Before making any changes, always read `PROJECT_BRIEF.md` first.

Follow the project brief carefully.

## Main Rules

- Build a clean, modern, premium Chiang Mai rental service website.
- Use Thai as the main language.
- Use Noto Sans Thai as the main font.
- Keep the design minimal, trustworthy, and easy to read.
- Make the website mobile-first.
- Use reusable components.
- Do not overcomplicate the first version.
- Focus on lead generation, zone-based property browsing, and property cards.

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS

## First Version Scope

- Use local sample data only.
- Do not connect Supabase yet.
- Do not build authentication yet.
- Build the frontend structure first.
