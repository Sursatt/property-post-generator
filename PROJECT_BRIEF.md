# Project Brief: Chiang Mai Rental Service Website

## Project Overview

This project is a clean, modern, premium rental service website for Chiang Mai. The website helps tenants find rental homes, condos, townhomes, and pool villas across Chiang Mai, while also helping property owners rent out their properties.

The website should feel international, trustworthy, easy to use, and suitable for both Thai and foreign customers looking for long-term rental homes in Chiang Mai.

## Main Business Goals

- Help tenants find rental homes, condos, townhomes, and pool villas in Chiang Mai.
- Help property owners rent out their properties.
- Organize rental homes clearly by Chiang Mai zones.
- Make it easy for tenants to submit their housing requirements.
- Make it easy for owners to submit properties for rent.
- Present the service as professional, reliable, and premium.

## Target Users

- Thai tenants looking for rental homes in Chiang Mai.
- Foreign tenants and expats looking for rental homes in Chiang Mai.
- Families looking for homes near international schools.
- Tenants looking for pet-friendly homes.
- Property owners who want help renting out homes, condos, townhomes, or pool villas.

## Design Direction

The website should use a mobile-first responsive layout with a clean and premium feel.

Design style:

- Clean
- Modern
- International
- Premium
- Trustworthy
- White background
- Black text
- Soft gray sections
- Deep green accent
- Clear spacing
- Readable typography
- Simple navigation
- High-quality property presentation

Avoid decorative, playful, overly colorful, or cluttered styling. The website should feel calm, polished, and professional.

## Typography

Main font:

- Use `Noto Sans Thai` as the main font.
- Use `Inter` for English text if needed.
- Do not use decorative fonts.
- Typography should be clear, readable, and suitable for both Thai and English content.

## Color Direction

Suggested visual direction:

- Background: white
- Text: black or near-black
- Section backgrounds: soft gray
- Accent color: deep green
- Supporting borders: light gray

The overall design should remain bright, clean, and premium.

## Main Navigation

The main navbar should include:

- หน้าแรก
- บ้านเช่าตามโซน
- ฝากโจทย์หาบ้าน
- ฝากบ้านให้ปล่อยเช่า
- เกี่ยวกับเรา
- ติดต่อเรา

The zone dropdown under `บ้านเช่าตามโซน` should include:

- นิมมาน
- สันติธรรม
- แม่เหียะ
- หางดง
- สารภี
- สันกำแพง

- แม่ริม
- ใกล้โรงเรียนนานาชาติ
- บ้านเลี้ยงสัตว์ได้

## Homepage Structure

The homepage should include these sections:

1. Hero section
2. Services section
3. Browse by zone section
4. Featured rental homes
5. How it works
6. Tenant call-to-action
7. Owner call-to-action

## Services

The services section should communicate these core services:

- ช่วยหาบ้านเช่า
- ช่วยปล่อยบ้านเช่า
- คัดบ้านตามโซน
- นัดดูบ้าน
- แนะนำทำเล
- บ้านสำหรับต่างชาติ

## Content Direction

The tone should be professional, warm, and trustworthy. The website should explain that the service helps tenants save time by matching them with suitable rental homes by location, budget, lifestyle, and property type.

For property owners, the website should communicate that the service helps present properties professionally, find qualified tenants, arrange viewings, and support the rental process.

## Property Types

The website should support these rental property types:

- บ้านเช่า
- คอนโดเช่า
- ทาวน์โฮมเช่า
- Pool villa
- บ้านใกล้โรงเรียนนานาชาติ
- บ้านเลี้ยงสัตว์ได้

## First Version Scope

For the first version:

- Do not connect Supabase yet.
- Do not build a complex backend yet.
- Use local sample property data first.
- Build clean frontend pages first.
- Focus on responsive layout, polished UI, and clear content structure.
- Keep the codebase easy to extend later when database and backend features are added.

## Initial Technical Direction

- Use Next.js App Router.
- Use TypeScript.
- Use Tailwind CSS.
- Keep reusable components simple and clear.
- Store initial sample property data locally in the project.
- Build pages and sections in a way that can later connect to Supabase or another backend.

## Success Criteria For Version 1

Version 1 should feel like a polished rental service website, not a rough placeholder. It should clearly show:

- What the business does
- Who it helps
- Which Chiang Mai zones are covered
- What services are offered
- Featured rental homes from local sample data
- Clear calls to action for tenants and owners

The website should be easy to browse on mobile and desktop, with a premium and trustworthy first impression.
